# Pacote: Interface Gráfica (View)

&nbsp;&nbsp;&nbsp;Este pacote contém todas as classes relacionadas à apresentação visual do sistema utilizando a biblioteca *Java Swing* e *AWT*.

## Principais Responsabilidades
* *Renderização:* Desenhar as imagens, barras de status e fundos na tela.
* *Interação:* Capturar eventos de clique do mouse e botões.
* *Feedback:* Exibir mensagens de erro ou sucesso para o usuário.
* *Animação:* Controlar transições visuais (sprites e efeitos).

## Classes
* *JanelaPrincipal:* O container principal (JFrame) da aplicação.
* *PainelTamagotchi:* Painel customizado para renderização avançada de imagens e GIFs.
* *BarraStatus:* Componente reutilizável para exibir atributos numéricos visualmente.
