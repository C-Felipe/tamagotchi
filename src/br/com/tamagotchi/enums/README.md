# Pacote: Enumerações (Constants)

Este pacote contém tipos enumerados (Enum) para garantir a integridade dos dados e evitar o uso de "Magic Strings" ou códigos numéricos soltos no código.

## Conteúdo
* *Humor:* Define estritamente os estados emocionais possíveis (FELIZ, TRISTE, MORTO, etc.), facilitando a lógica condicional na Interface Gráfica.
