# Pacote: Recursos (Assets)

Este diretório armazena todos os arquivos de mídia externos utilizados pelo sistema.

## Estrutura
* **/imagens:** Contém os sprites dos Digimons, cenários, ícones de botões e GIFs.
* **/sons:** Contém os efeitos sonoros (ex: som de evolução em `.wav`).

*Nota: Estes arquivos são carregados dinamicamente pelas classes do pacote `gui`.*
